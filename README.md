### README

- Nacos [服务注册中心与统一配置中心]
- Gateway [服务网关路由配置]
- Openfeign [服务调用负载均衡]
- Sentinel [服务熔断限流]
- Sleuth [链路追踪]
- Mybatis [持久层框架]
- Mysql [数据库]
- Docker [容器化项目部署]
- OSS [阿里云对象存储]
